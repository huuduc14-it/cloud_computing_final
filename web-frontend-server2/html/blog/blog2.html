☁️

1. Ba mô hình dịch vụ Cloud

Mô hìnhVí dụNgười dùng quản lý
IaaS – Infrastructure as a ServiceAWS EC2, Azure VMOS, Runtime, App, Data
PaaS – Platform as a ServiceHeroku, Google App EngineApp, Data
SaaS – Software as a ServiceGmail, SalesforceChỉ sử dụng

2. Mô hình triển khai

  • Public Cloud: AWS, Azure, GCP – chia sẻ hạ tầng đa khách hàng.
  • Private Cloud: Hạ tầng riêng của tổ chức.
  • Hybrid Cloud: Kết hợp public + private.

3. MyMiniCloud = Private Cloud thu nhỏ

Hệ thống 9 server trong dự án này mô phỏng một Private Cloud hoàn chỉnh với Web Hosting, App Backend, Database, Auth, Storage, DNS, Monitoring và API Gateway – tất cả kết nối qua Docker Network cloud-net.
