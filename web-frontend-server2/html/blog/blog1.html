ContainerDockerDevOps

📅 01/01/2025 · Tran Van Huy

🐳

1. Docker là gì?

Docker là nền tảng mã nguồn mở giúp tự động hóa việc triển khai ứng dụng bên trong các container phần mềm. Thay vì cài đặt trực tiếp lên máy chủ, ứng dụng được đóng gói cùng toàn bộ dependency vào một container độc lập.

2. Container vs Virtual Machine

Container nhẹ hơn VM vì chia sẻ kernel của OS host, không cần cài OS riêng → khởi động nhanh hơn, tiêu tốn ít RAM hơn.

3. Lệnh Docker cơ bản

docker pull nginx              # Tải image
docker run -d -p 80:80 nginx   # Chạy container
docker ps                      # Xem container đang chạy
docker compose up -d           # Khởi động toàn bộ stack
docker compose logs -f         # Xem log realtime

4. Ứng dụng trong MyMiniCloud

Trong dự án này, mỗi trong 9 server được đóng gói thành một Docker container riêng biệt, kết nối với nhau qua network cloud-net. Docker Compose quản lý toàn bộ vòng đời của hệ thống.
